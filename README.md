# Mudabbir - Personal Finance Dashboard

## Overview

Mudabbir is a responsive personal finance dashboard built using HTML and CSS. The project provides a clean and organized interface for tracking income, expenses, budgets, bills, payment methods, and account summaries in one place.

The dashboard is designed to help users gain better control over their finances through clear data visualization and structured financial management tools.

## Features

- Responsive dashboard layout
- Sidebar navigation
- Financial summary cards
- Recent activities and transactions section
- Monthly bills tracker
- Budget overview with progress indicators
- Income vs. expenses chart
- Payment methods management section
- Account summary section
- Interactive 3D credit card flip animation
- Mobile-friendly design

## Technologies Used

- HTML5
- CSS3
- CSS Grid
- Flexbox
- CSS Variables
- Media Queries

## Concepts Applied

- Responsive Web Design
- CSS Grid Layout
- Flexbox Layout System
- CSS Custom Properties (Variables)
- Semantic HTML
- CSS Positioning
- Pseudo-classes and Pseudo-elements
- CSS Transitions and Animations
- 3D CSS Transforms

## Learning Outcomes

Through this project, I practiced building a complete financial dashboard from scratch using vanilla CSS while improving my understanding of CSS Grid, responsive layouts, reusable styling with CSS variables, and modern UI design techniques.
